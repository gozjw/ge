# ge

### 文件加解密工具

加密后生成".ge"为后缀的加密文件，解密后生成"ge_"为前缀的原文件。

### 运行方式
1. 双击运行
2. 命令行，参数为文件或者文件夹

    - 加密
    ```
    ge.exe cs/1.jpg
    ```
    ```
    ge.exe cs/
    ```

    - 解密
    ```
    ge.exe cs/1.jpg.ge
    ```
    ```
    ge.exe cs/
    ```

    - 还原文件名
    ```
    ge.exe cs/ge_1.jpg
    ```
